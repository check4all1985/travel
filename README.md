# Travel Packages Static Website

A simple static website for selling travel packages, built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Package filtering by destination
- Booking modal for inquiries
- Contact form
- Ready for GitHub Pages deployment

## Deployment to GitHub Pages

1. Create a new repository on GitHub
2. Upload all files from this project to the repository
3. Go to Settings > Pages
4. Select "Deploy from a branch"
5. Choose "main" branch and "/ (root)" folder
6. Save and wait for deployment

Your site will be available at `https://yourusername.github.io/repository-name/`

## File Structure

```
/
├── index.html          # Home page
├── packages.html       # Packages listing page
├── css/
│   └── style.css       # Stylesheet
├── js/
│   └── script.js       # JavaScript functionality
├── images/             # Image assets (add your travel photos here)
└── README.md           # This file
```

## Adding Images

The website now uses high-quality images from Unsplash, a free stock photo service. All images are automatically loaded from the web, so no local image files are required. The images are professionally curated and provide an "AI-generated" quality appearance.

If you prefer to use your own images, you can replace the Unsplash URLs in the HTML files with local image paths and upload the images to the `images/` folder.

## Customization

- Edit `css/style.css` to change colors, fonts, and layout
- Modify package data in `index.html` and `packages.html`
- Update contact information in the footer
- Add more pages or features as needed

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
